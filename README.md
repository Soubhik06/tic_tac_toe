# Cypher Tic-Tac-Toe
A minimalist, dark-themed Tic-Tac-Toe game built with Godot 4.

## Deployment
**Live Demo:** [(https://soubhik06.itch.io/tictactoe)]

## Technical Details
- **Engine:** Godot 4.6.2 (Standard)
- **Language:** GDScript
- **Architecture:** Programmatic UI generation in a single scene for a "single source of truth."

## AI Disclosure
I used Gemini (AI) to generate the initial UI boilerplate and for troubleshooting Godot 4 web export configurations. The game logic and win-state verification were manually implemented/refined.
